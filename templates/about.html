About

I built this IDS dashboard to make security data easier to understand and act on.

Instead of digging through scattered logs, you can see live behavior, model output, and risk level together in one place.

What This Dashboard Does

It monitors incoming metrics, evaluates activity with a trained model, and reports whether behavior looks normal or suspicious.

The interface is designed to show useful signals first: attack type, confidence, and risk status.

Why I Built It

Most IDS demos look impressive but are hard to use day to day. I wanted something practical, understandable, and easy to test.

This project is my way of connecting machine learning with real security workflows, not just model accuracy numbers.

How You Can Use It

  • Watch live detection behavior and system health in real time.
  • Run simulated attacks to verify alerts and confidence changes.
  • Upload CSV activity data to analyze larger samples quickly.
  • Use it as a base for your own IDS experiments and improvements.
